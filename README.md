Imagens do site adaptado para Mobile.
<br>
<br>
Na versão mobile do site, foi implementado um menu interativo utilizando JavaScript para melhorar a navegação em telas menores. 
O menu é representado pelo ícone “hambúrguer”, que ao ser acionado exibe as opções disponíveis de forma organizada e acessível.

Essa abordagem permite economizar espaço na interface sem comprometer a usabilidade, já que os links de navegação ficam ocultos até que o usuário decida visualizá-los. 
O comportamento dinâmico garante uma experiência mais fluida e intuitiva, adaptada ao contexto mobile.
